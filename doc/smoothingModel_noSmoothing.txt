smoothingModel_noSmoothing command :h3

[Syntax:]

Defined in couplingProperties dictionary.

smoothingModel off; :pre

[Examples:]

smoothingModel off; :pre

[Description:]

The "noSmoothing" model is a dummy smoothingModel model which does no smoothing.
