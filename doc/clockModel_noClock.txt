clockModel_noClock command :h3

[Syntax:]

Defined in couplingProperties dictionary.

clockModel off; :pre

[Examples:]

clockModel off; :pre

[Description:]

The "noClock" model is a dummy clockModel model which does not measure/evaluate the run time.

[Restrictions:] none.
